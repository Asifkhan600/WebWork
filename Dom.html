Understanding console.dir()

Learn the difference between console.log() and console.dir()

What is console.dir()?

console.dir() is a method that displays an interactive list of the properties of a specified JavaScript object.

Example Usage:

const element = document.getElementById('myElement');
console.dir(element);
Output will appear here...

Difference from console.log()

While console.log() prints the element in an HTML-like tree, console.dir() displays the object as a JSON-like tree.

Try it yourself:

// Try both with a DOM element
console.log(document.body);
console.dir(document.body);
Output will appear here...

Key Differences

Feature console.log() console.dir()
DOM Elements Shows as HTML tree Shows as expandable object
Objects String representation Interactive properties list
Use Case General output Inspecting object properties
Format Varies by data type Always as object properties
